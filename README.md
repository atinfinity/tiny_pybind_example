# tiny_pybind_example

## Introduction

The <https://github.com/pybind/cmake_example> is example of pybind11 with a CMake-based build system.  
The `tiny_pybind_example` is a simplified version of `cmake_example` for pybind11 learning.

## Requirements

- CMake
- build-essential
- python3-dev
- python3-pip
- python3-pybind11
- NumPy
